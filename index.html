Data grid

Data - Edit

數據網格具有對單元格和行編輯的建置支持。

Data - Export

以各種文件格式（例如 CSV、Excel 或 PDF）輕鬆導出行。

Column painning

固定（或凍結、鎖定或粘性）列是在用戶水平滾動網格時始終可見的列。它們可以固定在左側或右側，並且不能重新排序。

Column Order

拖動列以重新排序

Column spanning

默認情況下，每個單元格佔用一列的寬度。您可以使用列跨越來修改此行為。它允許單元格跨越多個列。

Data - Selection

選擇允許用戶選擇並突出顯示他們欲執行或修改的行數。

Data - Filtering

根據一個或多個條件篩選行數據。

Column Sorting

根據一個或多個行進行排序。

Pagination

每頁包含 100 行。用戶可以通過頁腳中的選擇器來改變頁面的大小。

Tree Data

使用樹數據來處理具有父/子關係的行。
